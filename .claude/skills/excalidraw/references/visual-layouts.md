# Visual Layout Patterns

Use this reference when a diagram needs a composed layout rather than generic nodes and arrows.

## Soccer / Football Tactical Roster

Use a landscape canvas with:

- Title and subtitle at the top.
- Large green pitch on the left.
- Starter players as colored ellipses positioned by formation.
- Goalkeeper in yellow/orange, defenders in blue, midfielders in purple, attackers in green, featured player in red or gold.
- Bench/reserve panel on the right with position sections.
- Footer bar with group, next match, or source context.

Keep player labels short. If club names make text too long, split across lines or put club in smaller text.

## Column Journey

Use one column per stage. Each column should have:

- Stage number and title.
- 3 to 5 step cards.
- Arrows between columns.
- A footer with metrics or key checkpoints.

## Architecture Zones

Use grouped zones for:

- Users/channels.
- App/services.
- Data/integration.
- External platforms.
- Observability/security.

Prefer arrows that show important flows only. Avoid one arrow for every possible dependency.
